
# Source Code

This directory contains all standalone Java source files (`.java`) for the **01-Core-Java** repository.

Each program is designed to demonstrate a specific Core Java concept through a practical, executable example. These source files complement the Markdown documentation available in the repository by allowing learners to compile, execute, and experiment with the concepts directly.

---

# Purpose

The objective of this directory is to:

- Provide runnable Java programs for every Core Java topic.
- Demonstrate concepts through practical examples.
- Help beginners understand Java syntax and program structure.
- Support hands-on practice for students and self-learners.
- Assist in technical interview preparation.
- Serve as a quick reference for Java fundamentals.

# How to Run

Compile the Java program:

```bash
javac FileName.java
```

Run the compiled program:

```bash
java ClassName
```

Example:

```bash
javac 17-MultiplicationTableGenerator.java
java MultiplicationTableGenerator
```
