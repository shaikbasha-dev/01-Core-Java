# Rectangle Pattern in Java

## Introduction

The **Rectangle Pattern** is one of the most important pattern programs in Java. It extends the concepts learned in the Square Pattern by introducing **different numbers of rows and columns**.

Unlike a square, where the number of rows and columns are equal, a rectangle allows them to be different. This helps learners understand how nested loops can be controlled independently, making it easier to solve real interview pattern questions.

Once you master rectangle patterns, you can confidently move on to triangles, pyramids, diamonds, butterflies, and many other advanced patterns.

# Real-World Example

Imagine a classroom arranged with:

- **4 Rows**
- **6 Columns**

Since the number of rows and columns are different, the seating arrangement forms a rectangle.

Another example is a television screen, whiteboard, or notebook page, where the height and width are different.

Similarly, rectangle patterns print a fixed number of rows and columns that are not necessarily equal.

# Common Logic

Every rectangle pattern follows the same basic structure.

The outer loop controls the rows.

The inner loop controls the columns.

The only difference between patterns is the value printed inside the inner loop.

---

# Generic Algorithm

### Step 1

Read the number of rows.

### Step 2

Read the number of columns.

### Step 3

Repeat for every row.

### Step 4

Repeat for every column.

### Step 5

Print the required value.

### Step 6

Move to the next line after completing one row.

# Difference Between Square and Rectangle

| Square Pattern | Rectangle Pattern |
|---------------|-------------------|
| Rows = Columns | Rows and Columns can be different |
| Forms a square | Forms a rectangle |
| Uses one size value | Uses separate row and column values |
