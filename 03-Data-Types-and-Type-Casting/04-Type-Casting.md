Type Casting:
    Type casting is the process of converting a value from one data type to another.

There are two types of type casting in Java:
1. Implicit type casting
2. Explicit type casting

1. Implicit Type Casting:
    Implicit type casting is performed automatically by the Java compiler when a smaller data type is converted to a larger data type.
    This is also called widening conversion.
Example:
    int to float

2. Explicit Type Casting:
    Explicit type casting is required when converting a larger data type to a smaller data type.
    The programmer must explicitly tell the compiler to perform the conversion using a cast operator.
Example:
    long to int
