# 📖 About This Repository

This repository has been created as a **complete Core Java learning resource** that helps learners understand Java from the fundamentals to interview-oriented concepts.

Unlike repositories that only contain source code, this repository focuses on **conceptual understanding**, **hands-on programming**, and **technical interview preparation**.

Every topic is organized in a structured manner to help learners progress from beginner-level concepts to advanced programming confidence.

The repository includes carefully prepared notes, practical examples, code explanations, outputs, pseudocode, and interview questions to make learning easier and more effective.

Whether you are learning Java for the first time, revising concepts before interviews, or strengthening programming fundamentals, this repository serves as a structured reference throughout your learning journey.

# 🚀 How to Use This Repository

This repository follows a structured learning approach designed to help you build a solid understanding of Core Java concepts.

To get the best learning experience, follow the recommended workflow below.

```text
Read Theory
      │
      ▼
Understand the Concept
      │
      ▼
Study Example Programs
      │
      ▼
Run the Programs
      │
      ▼
Observe the Output
      │
      ▼
Modify the Code
      │
      ▼
Solve Similar Problems
      │
      ▼
Revise Interview Questions
```

### Recommended Learning Process

1. Start with **Java Basics**.
2. Read the theoretical explanation carefully.
3. Understand the concept before writing code.
4. Study the provided example programs.
5. Execute the programs on your local machine.
6. Compare the output with the expected result.
7. Modify the code to experiment with different scenarios.
8. Solve additional practice problems.
9. Revise the interview questions after completing each module.

Following this approach will improve both conceptual understanding and practical programming skills.

# 💡 Practice Exercises

The following exercises are recommended after completing each module.

## Java Basics

- Explain the difference between JVM, JDK, and JRE.
- Draw the Java compilation and execution process.
- Describe Java's platform independence.

---

## Basic Programs

- Write a program to calculate the factorial of a number.
- Reverse a number.
- Check whether a number is prime.
- Generate the Fibonacci series.
- Swap two numbers using different approaches.

---

## Data Types

- Demonstrate implicit type casting.
- Demonstrate explicit type casting.
- Compare primitive and non-primitive data types.

---

## Methods

- Create methods with different parameter combinations.
- Implement method overloading.
- Build a simple calculator using methods.

---

## Control Statements

- Print multiplication tables.
- Generate star patterns.
- Solve menu-driven programs using switch.
- Implement nested loops.

---

## Arrays

- Find the largest element.
- Find the smallest element.
- Reverse an array.
- Sort an array.
- Search an element using Linear Search.

---

## Strings

- Reverse a string.
- Count vowels and consonants.
- Check palindrome strings.
- Count word frequency.
- Remove duplicate characters.
